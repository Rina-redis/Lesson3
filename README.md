5. Тепер нам потрібно створити скріпт

Трохи теорії
```
Скріпт (англ. script – сценарій) – це невелика програма, яка містить послідовність дій, створених для автоматичного виконання завдання. 

Скріпт зберігається у вигляді текстового файлу на хостингу та доступний для запуску користувачами.
```
