### Наприклад
workspace.AnchoredBox.Anchored

Ми звертаємося до нашого блока AnchoredBox і потім до його властивості Anchored

```
Властивість Anchored визначає, чи буде частина нерухомою за фізикою. Якщо її ввімкнено, деталь ніколи не змінюватиме положення через силу тяжіння, зіткнення інших частин, перекриття інших частин або будь-які інші причини, пов’язані з фізикою. Частина, яка не закріплена, називається незакріпленою .
```

Призначаючи true(правда), ми закріплюємо блок у просторі

Логічно що завдяки Material ми обираємо матеріал для блока

```
Властивість Material дозволяє  встановлювати текстуру деталі та фізичні властивості. Стандартний пластиковий матеріал має дуже легку текстуру, а матеріал SmoothPlastic взагалі не має текстури. Деякі текстури матеріалів, як-от DiamondPlate і Granite, мають дуже помітні текстури. Текстура кожного матеріалу по-різному відбиває сонячне світло, особливо фольга.
```
